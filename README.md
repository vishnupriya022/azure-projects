### Azure Projects Portfolio ☁️

This repository contains beginner-friendly Azure projects using the Azure Free Tier.  
All projects include step-by-step instructions and screenshots to demonstrate hands-on experience.

### Projects List

1. ![Project 1 – Resource Group & Virtual Machine](Project-1-Resource-Group-VM)  
   Create a resource group and deploy a virtual machine.

2. ![Project 2 – Start, Stop & Delete VM](Project-2-Start-Stop-Delete-VM)  
   Learn to manage VM lifecycle: start, stop, and delete resources safely.

3. ![Project 3 – Storage Account](Project-3-Storage-Account)  
   Create Azure Storage, Blob containers, upload files, and manage resources.

4. ![Project 4 – VNet & Subnet](Project-4-VNet-Subnet)*
   Create a virtual network and subnet to organize cloud resources.

5. ![Project 5 – Monitor & Alerts](Project-5-Monitor-Alerts)  
   Monitor VM metrics, create alerts, and receive notifications.
